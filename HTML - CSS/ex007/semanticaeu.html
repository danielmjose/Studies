Negrito/Destaque

Itálico/ênfase

Para escrever em Itálico é encessário escrever a taeg <em> que significa Emphasis!

Marcador!

Para você grifar algo como se usasse um marcador é necessário escrever <mark>

Texto menor!

Para escrever um texto pequeno, é necessário escrever a tag <small>

Texto cortado!

Para escrever um texto cortado é necessário escrever<strike> apesar de eu achar que talves entre em desuso(?)

Texto inserido ou sublinhado

Sobreescrito!

Subscrito!
